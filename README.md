# ⚽ FIFA World Cup Analytics Dashboard

A data analytics project analyzing FIFA World Cup matches using Python and Streamlit.

## 🚀 Features
- Interactive dashboard
- Team performance analysis
- Match result distribution
- KPI metrics
- Win prediction model (ML)

## 🛠️ Tech Stack
- Python
- Pandas
- Streamlit
- Matplotlib
- Scikit-learn

## 📊 Dataset
FIFA World Cup historical match data
